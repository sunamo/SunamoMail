# NuGet Alternatives to SunamoMail

This document lists popular NuGet packages that provide similar functionality to SunamoMail.

## Overview

Email operations

## Alternative Packages

### MailKit
- **NuGet**: MailKit
- **Purpose**: Modern email library
- **Key Features**: SMTP, IMAP, POP3, MIME, OAuth2

### FluentEmail
- **NuGet**: FluentEmail.Core
- **Purpose**: Fluent email API
- **Key Features**: Templates, attachments, multiple senders

### MimeKit
- **NuGet**: MimeKit
- **Purpose**: MIME message parsing
- **Key Features**: Email parsing, construction, S/MIME, PGP

### System.Net.Mail
- **NuGet**: System.Net.Mail
- **Purpose**: Built-in SMTP (legacy)
- **Key Features**: Basic SMTP (not recommended)

## Comparison Notes

MailKit is the gold standard for email in .NET. FluentEmail for simplified API.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **MailKit**: Modern email library
- **FluentEmail**: Fluent email API
- **MimeKit**: MIME message parsing
